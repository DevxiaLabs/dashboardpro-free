# DashBoard Free

A modern, elegant dashboard template built with Next.js 14+, Tailwind CSS, and TypeScript. This is the free, open-source edition of DashBoard Pro — perfect to explore the design system and core layout.

Upgrade to Pro to unlock 30+ pages and 15+ advanced components.

## What's Included (Free)

- Authentication: Login
- Dashboard Home
- Settings: Profile, Appearance (dark/light + color themes)
- UI Components (essentials): Buttons, Inputs & Forms, Selects, Checkboxes & Toggles, Tabs, Cards, Badges
- Dark/Light Mode, Color Themes, Responsive Layout

## What's in Pro

- Pages: Analytics, Users, Products, Orders, Messages, Calendar, Kanban, Pricing, Invoices, Profile, FAQ, Roles & Permissions, Maps, Timeline, Files, Notifications, Changelog, Onboarding, i18n, Accessibility, Print Views, Landing, and more
- Advanced Components: Date Picker, Range Slider, Data Table, Modals, Drawers, Toasts, Tooltips, Accordion, Breadcrumbs, Progress & Spinners, Stepper, Avatar Group, Timeline, Pagination, Rich Text Editor, File Upload, Tree View, Color Picker, Charts

## Tech Stack
- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS 3.4+
- Lucide Icons

## Getting Started

Prereqs: Node.js 18+

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build
npm run build
```

Open http://localhost:3000 in your browser.
